# 🚀 איך להריץ את ResolveAI

## דרישות מקדימות

וודא שיש לך Python 3.8 ומעלה מותקן על המחשב שלך.

## התקנה והרצה

### שלב 1: Clone הפרויקט (אם עדיין לא עשית)

```bash
git clone https://github.com/yanaydavid/ResolveAI.git
cd ResolveAI
```

### שלב 2: התקן את התלויות

```bash
pip install -r requirements.txt
```

אם אין קובץ `requirements.txt`, התקן ידנית:

```bash
pip install streamlit reportlab python-docx PyPDF2
```

### שלב 3: הרץ את האפליקציה

```bash
streamlit run app.py
```

### שלב 4: פתח את הדפדפן

האפליקציה תיפתח אוטומטית בדפדפן שלך בכתובת:
```
http://localhost:8501
```

## 🎯 איך להשתמש באפליקציה

### פורטל תובעים 🏛️

1. **מלא פרטים אישיים**
   - שם מלא
   - כתובת מייל
   - מספר טלפון נייד

2. **העלה כתב תביעה**
   - PDF או Word (.docx)

3. **מלא פרטי נתבע**
   - שם מלא של הנתבע
   - מספר טלפון נייד של הנתבע

4. **אשר תקנון**
   - קרא והסכם לתנאי השימוש
   - אשר שליחת דואר רשום

5. **הגש תביעה**
   - קבל מספר תיק ייחודי
   - הנתבע יקבל SMS עם הקישור

### פורטל נתבעים 🛡️

1. **עבור ל"פורטל נתבעים"** בסרגל הצד

2. **הזן מספר תיק**
   - המספר שקיבלת ב-SMS

3. **השלם רישום**
   - מלא את הפרטים האישיים שלך
   - אמת את המספר טלפון

4. **הורד את כתב התביעה**
   - לחץ על כפתור ההורדה

5. **העלה כתב הגנה**
   - PDF או Word (.docx)
   - הגש את התשובה שלך

6. **קבל פסק בוררות**
   - המערכת תנתח אוטומטית
   - הורד את פסק הבוררות ב-PDF

## 📄 מה כולל פסק הבוררות?

✅ **קוד אימות (Hash)** - SHA-256 לאימות שלמות המסמך
✅ **חותמת זמן** - תאריך ושעה מדויקים
✅ **כותרת משפטית** - "פסק בורר לפי חוק הבוררות, התשכ"ח-1968"
✅ **ניתוח נקודות מחלוקת** - טבלה מפורטת
✅ **הצהרת אובייקטיביות** - הסבר על הניתוח ה-AI
✅ **נספח אימות** - דף הסבר לשופט
✅ **סיכום כספי** - סכומים ותאריכי תשלום

## 🐛 פתרון בעיות

### האפליקציה לא עולה?

1. וודא ש-Python מותקן:
   ```bash
   python --version
   ```

2. וודא שכל התלויות מותקנות:
   ```bash
   pip list | grep streamlit
   pip list | grep reportlab
   ```

3. אם יש שגיאות, נסה:
   ```bash
   pip install --upgrade streamlit reportlab python-docx PyPDF2
   ```

### שגיאת fonts בעברית?

זה תקין - המערכת תשתמש ב-fallback font. הטקסט יוצג נכון.

### בעיות עם database?

המערכת יוצרת את ה-database אוטומטית בקובץ `resolve_ai.db`
